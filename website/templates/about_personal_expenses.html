About Personal Expenses dataset

The Personal Expense dataset used in this study was sourced from Kaggle (Chauhan, 2023). It contains Sanjay Chauhanâ€™s personal expense records collected in March 2023. The dataset documents what he ate at different times of the day and how much money he spent on various items. Its primary purpose is to analyze spending behavior and food-related expenses. The dataset includes 144 rows and 6 variables.

Description of Final Dataset

Name Data Type Description
Date Date Date of the event
Item Object (String) Item for eating, drinking, clothing or other
Amount Float How much amount of item
Category Object (String) Alone or with friend
Time Date Time of shopping
day Object (String) Name of the day of the event
